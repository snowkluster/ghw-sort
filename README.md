## Start

To start the application 

```sh
./start.sh
```

OR

```sh
chmod +x start.sh \
./start.sh
```

This will start the flask app at [localhost:5000](http://localhost:5000)

## Stop

To stop the applicaton

```sh
./start.sh stop
```

simply pass the `stop` argument to the start script
